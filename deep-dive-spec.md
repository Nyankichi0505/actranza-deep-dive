# 深掘り項目・判定基準 仕様書

## 5つの深掘り項目

### ① 投与経路・デリバリー方式の詳細
- 注射経路（皮下/筋肉/皮内/静脈/腫瘍内/吸入/経口など）
- 現在使用されている注射器・デバイスの種類
- **無針化の余地評価**（用量・粘度・投与頻度・適応から判定）

### ② 開発企業・研究者プロファイル
- リード研究者（PI）・所属機関・直近論文
- 企業の場合：開発ステージ・パイプライン全体・研究組織
- ActranzaLab見込客になりうる研究グループの具体的リスト

### ③ 自社接点・営業仮説（ActLab／Daicel別）
- ActranzaLab（前臨床研究用）の使用見込み・営業仮説
- ダイセルインジェクター（臨床用）の使用見込み・営業仮説

### ④ 競合デリバリー技術の動向
- 競合無針注射技術（PharmaJet, Inovio, Portal Instruments, Vaxxas 等）
- 代替技術（マイクロニードル・経皮・吸入）の進捗
- 自社技術の差別化ポイント

### ⑤ 国内連携可能性
- 日本人研究者の関与・国内臨床試験
- 国内企業（武田・第一三共・塩野義・国内アカデミア）の関連動向
- 国内パートナー候補

## Heat Score 判定基準

### ActLab接点度（前臨床研究での使用見込み）

| 判定 | 条件 |
|------|------|
| **A** | 前臨床/基礎研究 + 核酸/タンパク/ペプチド/ワクチン抗原 + 皮内/皮下/筋肉内投与 |
| **B** | 臨床ニュースだが前臨床研究も活発な領域 / 同モダリティで他研究者の論文・前臨床が増えている / 競合企業も前臨床段階のものを持っている |
| **C** | 小分子・抗体薬で注射経路特有のメリットが薄い / 経口・吸入・点滴等が中心で皮内/皮下/筋肉内の余地がない / 完全な臨床応用フェーズで研究的バックトラックの可能性が低い |

### Daicel接点度（臨床応用での使用見込み）

| 判定 | 条件 |
|------|------|
| **A** | ワクチン（特に大量接種シナリオ）/ 皮下投与の生物製剤（GLP-1, mAb 等）/ 自己投与・在宅医療向け / 既に無針注射技術と組み合わせた臨床試験が進行中 |
| **B** | 皮内/皮下/筋肉内投与の生物製剤だが現状は通常注射器で開発中（**無針化の余地あり**）/ 高頻度投与が必要で自己投与化メリットあり |
| **C** | IV点滴・徐放注射・脳室内・関節内投与 / 投与頻度が極めて低く（年1回など）無針化メリットが薄い / 経口・吸入で完結する治療 |

## 両軸独立判定の例

| ニュース | ActLab | Daicel | 解釈 |
|---------|--------|--------|------|
| Moderna mRNAがんワクチン Ph3結果 | B | A | 前臨床研究は他者がやってる／無針臨床応用の最有望先 |
| Intellia in vivo CRISPR Ph3 | B | C | 前臨床は活発／IV・LNP送達なので臨床無針は対象外 |
| 東大の前臨床マウス（mRNA皮内） | A | B | ActLab直接顧客／将来臨床化で無針候補 |
| Ionis ASO（B型肝炎）Ph3 | C | C | 完成薬剤・既存技術で十分 → **対象外（レポート省略）** |
| 国内アカデミアのDNAワクチン基礎研究 | A | B | 直接顧客／臨床化したら国内パス先 |

## ロック・リトライ

- 完了時に `完了` または `対象外` に変更
- 処理中のままになっている記事はクラッシュなどの可能性 → 手動でリセットが必要
