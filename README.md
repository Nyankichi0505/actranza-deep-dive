# actranza-deep-dive

Notion「📰 HP/News記事」DBの「ステータス＝興味あり」記事を深掘り調査し、ActLab接点度・Daicel接点度を判定して詳細レポート子ページを生成する自動化ジョブ。

## ビジネス目的

ActranzaLab（理化学機器・無針注射器・研究用）のマーケ担当が、興味のあるバイオ医療ニュースから:

1. **マーケ視点の市場・トレンド把握**
2. **ActranzaLab見込客（研究者）の発掘**
3. **社内ダイセルインジェクターチーム（同仕組みの臨床用）への顧客パス候補判定**
4. **理化学機器・医療機器チーム連携用ナレッジ蓄積**

を毎日12:00 JST（1日1回）自動で行うのが目的。

## 動作

```
[Notion DB] ← ユーザーがステータスを「興味あり」に変更
   ↓ 毎日12:00 JSTにRoutineが検知
[Routine: actranza-news-deep-dive]
   ↓ 「深掘り」プロパティを「処理中」に
   ↓ WebSearch/WebFetch で5項目調査
   ↓ ActLab×Daicel接点度をA/B/C判定
   ├─ 両方C? → 親ページに判定だけ記録 + 「対象外」/ レポート省略
   └─ それ以外? → 子ページに詳細レポート + 「完了」
```

## 5つの深掘り項目

1. **投与経路・デリバリー方式**（無針化余地評価含む）
2. **開発企業・研究者プロファイル**（ActLab見込客発掘）
3. **自社接点・営業仮説**（ActLab/Daicel別）
4. **競合デリバリー技術動向**
5. **国内連携可能性**

## Heat Score 判定基準

### ActLab接点度（前臨床研究用途）
| 判定 | 条件 |
|------|------|
| A | 前臨床+核酸/タンパク/ペプチド/ワクチン抗原+皮内/皮下/筋肉内 |
| B | 臨床ニュースだが前臨床研究も活発な領域 |
| C | 小分子・抗体薬 / IV・経口・吸入中心 |

### Daicel接点度（臨床応用用途）
| 判定 | 条件 |
|------|------|
| A | ワクチン大量接種 / 皮下生物製剤 / 自己投与 / 既に無針併用試験あり |
| B | 皮内/皮下/筋肉内生物製剤だが現状通常注射器（無針化余地） |
| C | IV/徐放/脳室/関節内 / 年1回 / 経口・吸入完結 |

## ファイル構成

- `prompt.md` - Routine が毎日12:00 JSTに実行する指示文
- `deep-dive-spec.md` - 深掘り項目とHeat Score判定基準の詳細仕様
- `notion-schema.md` - 保存先 Notion DB のスキーマ
- `CHANGELOG.md` - 仕様変更履歴
